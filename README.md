## How to run this project

Clone this repo

```bash
  git clone https://github.com/Alex-dll/Api-Filmes
```

Go to project directory

```bash
  cd my-project
```

Install the dependencies

```bash
  npm install | yarn install
```

Start the aplication

```bash
  npm run dev | yarn dev
```
